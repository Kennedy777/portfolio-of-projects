Personal Website Project

This project was created with a bootstrap theme named Freelancer, combined with the work of Ian Lunn – Hover.css . Some modifications were made for personalization purposes, such as adding my picture, new sections, and different glyphicons.

With this project, I got in touch with Sass and started to get a feel on how to style pages with proper styling rules, although I need a lot more practice :D. I also learned how to configure a small php program to receive emails directly from a web form.

Diabete en Forme
(French)

This project was the second real world web project I contributed to. This project was developed as part of a funded study and is a web-based, personalized intervention promoting regular aerobic PA among adult Canadian francophones with type 2 diabetes. I developed the image of the site, a part of the business logic behind it, and the behaviour change system by which participants would increase their physical activity level.

Furthermore, this project has been a stepping-stone for me, because it made me fell in love with programming and its superpowers. It really kickstarted my self-driven programming education journey.
